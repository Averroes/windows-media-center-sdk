M - Windows Media Center ITV Application Sample

This is the sample Windows Media Center ITV Application. When launched this sample will display a mosaic of four streaming channels which the end user can navigate between.

This sample is intended to show some of the general capabilities of the iTV Framework APIs. The DevInstall.cmd script can be run from a command prompt with administrator privileges to perform a development install of the application. The Windows MediaCenter SDK talks about creating a windows installer package (*.msi) Setup program for final deployment.
